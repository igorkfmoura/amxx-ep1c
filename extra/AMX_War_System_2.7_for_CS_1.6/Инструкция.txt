И так...На CW сервера были установлен новый Мод AMX_WAR 2.7
Вот инструкция по управлению сервером для Модераторов
Всем внимательно изучить, чтобы потом вопросы глупые никто не задавал
Автором плагина является Grisbilen


Поддерживаются следующие режимы игр
Maxrounds, Timelimit, Winlimit

Дополнительные раунды(OverTime)
Если команды сыграли вничью, можно сыграть овертайм, или закончить матч

Выбор начальных сторон
-Команды сами договариваются за кого начинать
-Плагин бросает монетку и распределяет команды по результату монетки
-Команды играют раунд на ножах, выигравшая команда выбирает сторону

Матч на нескольких картах
Матч можно проводить на одной или двух картах по очереди

Режим разминки(Warmup)
Перед каждой половинкой игры запускается режим разминки, игроки должны написать say ready, чтобы подтвердить свою готовность играть, когда все игроки готовы, плагин делает рестарты и игра начинается
Можно установить ограничение на время разминки, после которого матч автоматически стартует

Восстановление игровой ситуации
Вылетевшим во время игры игрокам, при возвращении возвращаются деньги, броня, оружие, щипцы, фраги и смерти
Если сервер во время матча некорректно завершил работу(вылетел), после загрузки восстанавливается матч с раунда на котором он вылетел, после рестартов игрокам возвращаются деньги, броня, оружие, щипцы, фраги и смерти

Загрузка отдельных конфигов
Для кнайф раунда, разминки, матча, овертайма загружаются свои конфиги

Оповещение игроков
После каждого раунда выводится информационное сообщение о состоянии матча
Плагин начинает свою работу после старта команды админом
aw team1 team2 optional minready optional mrXX,mxXX,tlXX optional map=map1name optional map2=map2name optional pw=password optional cfg=cfgfile

Где
team1 - команда, которая начинает игру за Counter-Terrorist'ов
team2 - команда, которая начинает игру за Terrorist'ов
optional minready - Количество игроков в команде
optional mrXX,mxXX,tlXX - Режим игры 1)mrXX - Max Rounds, Первая половина XX раундов, вторая до победы одной из команд (выигрывает XX+1 раунд) 2)mxXX - Игра идет XX2 раундов, тоесть вторую половину играют до конца 3)tlXX - Time Limit

Поясню
ХХ это переменные команд mp_maxrounds (в режимах 1,2) и mp_timelimit (в режиме 3)
optional map1 - На какой играть карте, если параметр отсутствует, матч начнется на текущей карте
optional map2 - Если указать, то матч состоится на 2х картах
optional pw=password - установить пароль для входа на сервер на время матча, можно не задавать параметр
optional cfg=cfgfile - Какой конфиг для турнира загружать из папки ...cstrikeamxwarcfg , можно не задавать параметр, тогда используется default.cfg..его мы и не трогаем!

Отмечу
1) Параметры с пометкой optional не обязательны для запуска матча
2) Опциональные параметры можно вводить в любом порядке

например можно запустить матч так
aw SK Virtus.pro
1) тогда вместо minready, gametype встанут значения awc_minready_default и awc_gametype_default указанные в amxwaramxwar.cfg
2) при этом плагин проверит amxwarclans.ini, и если там не прописаны эти кланы, он их пропишет. Это нужно для меню плагина

и теперь команды
awstart – Начать матч, не дожидаясь всех ready
awabort – Прервать матч
awrestart – Перезапустить матч (текущую карту)
awrestart2 – Перезапустить половинку матча (карты)
swap – Поменять имена команд местами

ПРИМЕР ЗАПУСКА КВ
aw Team1 Team2  
Тут начинается этап разминки...
Если кланы готовы, то каждый игрок пишет ready или админ пишет awstart (в консоле) если хочет запустить КВ принудительно!

Запустить, отменить, автостартануть матч можно с помощью меню, для этого нужно набрать команду awmenu
